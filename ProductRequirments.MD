Take Home Prompt: You are working with a bank that needs to manage its liquidity. Build a simple full stack application that:

    Pulls data on treasury yields
    Plots the yield curve back to the user
    Allows the user to submit an order for a specific term and amount
    Displays the user’s historical order submissions

Please use libraries, languages, etc. of your choice. The prompt is deliberately open-ended to allow for your flexibility and preferences. Please do not spend more than 4-6 hours on this take-home.
 
The only specific requirements we have are:

    This should be uploaded to GitHub either as a public repo or a private repo shared with Alex (GitHub username: @admarko)
    The project should have a top-level readme.md file with instructions on how Alex can run
    Along with your GitHub link, please include a short video demo of your app. No audio is needed, just a ~30-second screen recording showing the app in action (clicking around to confirm it works). This helps our non-technical team review your work without running it locally. Feel free to use any screen recording tool you prefer.
